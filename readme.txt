AR Theme is a modular WordPress theme with responsive headers, hero slider, custom templates, and dynamic styling support.

== Description ==

AR Theme is a custom-built WordPress theme designed for flexibility and scalability. It uses a modular structure that allows developers to easily extend and maintain the codebase.

Features include:

- Responsive header system
- Mobile overlay and off-canvas menus
- Hero slider functionality
- Customizer integration
- Dynamic color system
- Widget and sidebar support
- Ajax search
- Reusable template parts
- Custom page templates
- Theme JSON support

Ideal for agencies, portfolios, and custom client projects.

== Installation ==

1. Upload the theme folder to `/wp-content/themes/`
2. Activate via Appearance > Themes
3. Configure settings in Customizer
4. Assign menus and widgets
